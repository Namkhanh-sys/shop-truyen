⚙️ Cấu Hình Gửi Gmail

MangaZone dùng EmailJS (miễn phí) để gửi email xác nhận đơn hàng.
Làm theo 3 bước dưới đây để kích hoạt:

Bước 1: Đăng ký miễn phí tại emailjs.com
Bước 2: Tạo Email Service (chọn Gmail) → lấy Service ID
Bước 3: Tạo Email Template với các biến: {{order_id}}, {{manga_title}}, {{volumes}}, {{date}}, {{total}}, {{to_email}} → lấy Template ID
Bước 4: Vào Account → API Keys → lấy Public Key
